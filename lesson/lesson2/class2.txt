ls指令
ls == list:列出当前目录的文件和目录。
ls -a:显示当前全部文件（包括隐藏文件）和目录。
	.当前目录
	..上级目录
	以点开头的文件就是隐藏文件。
	隐藏文件：这个文件如果没有特别要求你不需要操作它。
ls -l:显示当前目录文件和目录更详细的信息。
ls -d:显示目录本身的属性。不进入目录里面。

pwd指令
pwd = Print Working Directory:显示当前用户所在的路径。
路径	

cd指令
cd = change directory:切换路径。
cd ..:切换上级目录
cd -:切换热点路径切换,最近一次所处的路径。跳转到上一次的路径。
cd + 路径：切换到你想去的路径
cd ../.. 切换两级目录
cd ~:切换到当前用户的家目录。
~特定用户登录的时候默认路径。 /home/usr/, /root

window:也存在家目录。

相对于自身当前所处的路径，进行文件定位，相对路径。
相对路径简单。
绝对路径一般都是官方的配置文件的项目。

如何理解文件？
文件本身？
	创建一个空文件，占不占磁盘空间？必须占的！
	文件 = 文件的内容 + 文件的属性（元数据）
	文件的属性是数据吗？一定是数据。
	文件 == 文件内容数据 + 文件属性数据。
	学习操作文件的方法：操作文件内容的 + 操作文件属性的。
	
文件之间？（普通文件vs目录）
	普通文件
	目录文件--->文件夹--->保存其它文件的场所。
	
linux目录下：目录里面，可以再放目录吗？当然可以的。	
一个目录里面既可以是目录又可以是普通文件的。
几乎任何操作系统文件的目录组织结构是一颗二叉树。

mkdir指令
mkdir ==  make directory,创建一个新的目录
mkdir dir 创建一个空目录
mkdir -p a/b/c/d 递归创建一个目录
所以记住：mkdir -p = make directory with parents（创建目录及其父目录）！
