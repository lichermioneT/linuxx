  输入设备------存储器-------输出设备
                  |
                  |
                  |
                运算器-----中央

1.这里的存储器指的是谁?
  这里的存储器是内存。
  内存掉电易失。临时存储数据。
  
  外设：
  输入设备，输入设备都外部设备。
  磁盘和网卡既是输入设备也是输出设备
  外设相对于：内存 + CPU
  
  CPU == 运算器 + 控制器 + 其它

  CPU   快      计算的        需要数据
  内存  较快    临时存储      大大的缓存
  外设  较慢    永久存储
  
  CPU其实很笨，只能被动接受别人的指令，别人的数据-->执行别人的指令，计算别人数据的目的。
  CPU必须先，认识别人的指令？ CPU有自己的指令集。 我们写代码，编译本质-->二进制可执行程序。实际上就是翻译成CPU的指令集。
  CPU在读取和写入的时候，在数据层面，只和内存打交道。为了提高整机效率。

  内存里面不是天然就有数据的。
  开机的时候加载 操作系统加载到内存。
  
  操作系统--->帮我们做决策。
  
  内存
  I/O  读写就是IO的过程。
  外设
  
数据层面
CPU不和外设直接打交道，和内存直接打交道。
所以的外设，有数据需要载入，只能载入到内存中，内存写出，也一定是写到外设中

CPU不和外设直接打交道，只和内存直接打交道。
